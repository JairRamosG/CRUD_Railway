# FastAPI CRUD API con PostgreSQL y Railway

Esta plantilla es una API REST funcional construida con **FastAPI**, **SQLAlchemy Core** y **PostgreSQL**.  
Referencia para futuros proyectos de Forecasting:

- Operaciones CRUD completas (Crear, Leer, Actualizar, Eliminar) sobre una tabla `items`.
- Uso de SQLAlchemy Core para reflejar o definir esquemas de base de datos.
- Manejo de variables de entorno con `python-dotenv`.
- Health check listo para entornos cloud (Railway, Render, etc.).
- Configuración de despliegue en Railway con `railway.toml`.

---

## Tecnologías utilizadas

- **FastAPI** – Framework web moderno y rápido.
- **Uvicorn** – Servidor ASGI.
- **SQLAlchemy (Core)** – Interacción con la base de datos.
- **psycopg2-binary** – Adaptador PostgreSQL.
- **Pydantic** – Validación de datos y serialización.
- **Railway** – Plataforma de despliegue (opcional, pero incluida).
